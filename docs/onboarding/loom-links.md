## How to update

- **Who records:** William @ Brainstorm Projects, with DigimountAI on standby for any UI clarification.
- **Where the links go:** paste the Loom share URL into the URL column above. Update the Recorded date (YYYY-MM-DD) and Length (mm:ss).
- **Re-recording:** delete the old Loom in Loom's web UI, record a new one, paste the new URL into the same row. The in-app **?** link is unaffected — it points at the markdown one-pager, not at a Loom.
- **Adding a new walkthrough:** add a row to the relevant role's table; do not restructure the file. Plan 07-04 fills the existing six rows; future retainer scope can append rows as needed.
