<objective>
Plan 03-03 shipped a working but flawed UX: a 5-option modal with sales-pipeline jargon, a button that fires `invalid_status` on completed leads, and a `completed_today` counter that double-counts because it sums every `lead_events.type='call'` row including the `'connected'` event from `mark_lead_contacted`. This plan rewrites the agent surface to be drastically simpler — collapses Qualified/Won into a single **Converted** outcome, replaces the 5-option modal with **inline buttons on each card**, broadens the **Callbacks** tab into **Follow-ups** (covers callbacks + stalled no-answers), adds a **date-range selector** for stats/list scoping, and fixes the two bugs (counter + dead-button) at the database layer.

Purpose: this is the surface agents will live in 8h/day. It must be obvious to use, gamification-ready (Converted is the green anchor), and never expose impossible actions.

Output: 4-tab surface (To Call / Follow-ups / Converted / Lost), inline outcome buttons, date-range selector wired to `?range` URL, no-answer flow that increments `call_attempts` without flipping status, all RPC paths covered by Vitest, Playwright golden path rewritten to assert the new flow end-to-end.
</objective>

       DECLARE
         v_assigned_to  uuid;
         v_country_code text;
         v_jwt_country  text := (auth.jwt() ->> 'country_code');

         v_call_attempts integer;
       BEGIN
         SELECT assigned_to, country_code INTO v_assigned_to, v_country_code

         IF v_assigned_to IS DISTINCT FROM v_caller OR v_country_code IS DISTINCT FROM v_jwt_country THEN

         END IF;
         UPDATE public.leads
           SET call_attempts = call_attempts + 1, last_outcome = 'no_answer'
           WHERE id = p_lead_id
           RETURNING call_attempts INTO v_call_attempts;
         INSERT INTO public.lead_events (lead_id, actor_id, type, outcome)
           VALUES (p_lead_id, v_caller, 'call', 'no_answer');
         RETURN jsonb_build_object('lead_id', p_lead_id, 'call_attempts', v_call_attempts);
       END;
       $$;
