**4. Add DAL functions** in `packages/supabase/src/dal/country.ts`. First line: `import 'server-only';`. Use the cookie-authed server client from `../server` (matches `dal/queue.ts` pattern). RLS does the country lock — never service-role on reads.

Functions:
- `getCountryTodayStats(country_code: string)` — `SELECT * FROM country_today_stats WHERE country_code = $1` (single row)
- `getCountryStatsInRange(country_code: string, from: string, to: string)` — calls `country_stats_in_range` RPC
- `getLeadsByServiceToday(country_code: string)` — `SELECT * FROM leads_by_service_today WHERE country_code = $1 ORDER BY leads_count DESC` (multi-row)
- `getStatusPipelineToday(country_code: string)` — `SELECT * FROM status_pipeline_today WHERE country_code = $1` (multi-row, one per status)
- `getCountrySpeedToLeadToday(country_code: string)` — `SELECT * FROM country_speed_to_lead_today WHERE country_code = $1` (single row)
- `getAgentPerformanceInRange(country_code: string, from: string, to: string)` — calls `agent_performance_in_range` RPC, returns `AgentPerformanceRow[]`
- `getSpeedToLeadSeries(country_code: string, from: string, to: string)` — calls `speed_to_lead_series` RPC, returns `SpeedToLeadDay[]`
- `getCountryAgents(country_code: string)` — `SELECT user_id, full_name FROM user_roles WHERE role = 'sales_rep' AND country_code = $1 ORDER BY full_name` (drives the reassign-dialog dropdown later in 04-03)
- `reassignLead(input: ReassignLeadInput)` — calls `reassign_lead` RPC, returns `void`. Maps Postgres error codes to friendly errors: `42501 → ForbiddenError`, `P0002 → NotFoundError` (mirror the error-mapping pattern in `dal/queue.ts`).

Re-export from `dal/index.ts`.

**Don't compute aggregations in TypeScript** — every aggregation lives in the views/RPCs from 04-01. The DAL is a thin SDK wrapper, identical to `dal/queue.ts`.

**7. `_components/speed-to-lead-chart.tsx`** ("use client") — Recharts `AreaChart` wrapper. Pattern verified in RESEARCH.md (Context7 + AMA TrendChart fusion):
- Wrap in `<div className="h-12">` so `<ResponsiveContainer width="100%" height="100%">` doesn't collapse to 0×0 (RESEARCH.md pitfall 2).
- `<AreaChart data={data} margin={{ top: 4, right: 0, left: 0, bottom: 0 }}>`
- Linear gradient fill (`stopColor="#10b981"`, opacity 0.15 → 0.02).
- `<ReferenceLine y={300} stroke="#94a3b8" strokeDasharray="3 3" />` — the 5-minute target threshold (300 seconds; the DB stores seconds).
- `<Area type="monotone" dataKey="median_seconds" stroke="#10b981" strokeWidth={2.5} fill="url(#speedGrad)" isAnimationActive={false} />` — disable animation to avoid SSR hydration jitter (RESEARCH.md pitfall 1).
- Empty data → render `<div className="h-12 flex items-center text-xs text-slate-400">No data</div>` (don't render an empty chart).

**8. `_components/use-country-broadcast.tsx`** ("use client") — typed wrapper around `usePrivateBroadcast<T>` from `@repo/supabase/realtime`. Exact same shape as `apps/web/app/(sales-rep)/_components/use-agent-broadcast.ts` but topic is `` `country:${country_code}` `` and event is `'*'` (matches the broadcast trigger from migration 00008 + Phase 3 plan 03-02 carry-forward — listening on `'*'` because the webhook path emits `UPDATE` not `INSERT`).

```typescript
"use client";
import { usePrivateBroadcast } from "@repo/supabase/realtime";
import type { Database } from "@repo/supabase/types";

type LeadRow = Database["public"]["Tables"]["leads"]["Row"];

export function useCountryBroadcast(country_code: string, onChange: (row: LeadRow) => void) {
  return usePrivateBroadcast<LeadRow>({
    topic: `country:${country_code}`,
    event: "*",
    onMessage: onChange,
  });
}
```

**Don't fetch in `useEffect`** — every component receives its data via props from the Server Component. The broadcast hook is the ONLY client-side data source, and even it just optimistically bumps a counter.

**Don't add a chart wrapper component** — `SpeedToLeadChart` is the only Recharts importer; pages compose it directly.

**Don't filter by `country_code` in client code** — RLS does the lock; a TS filter is dead code.
